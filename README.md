# Free C

下载链接：
[v1.0.0 latest](https://github.com/StarrySky-skyler/FreeC/releases/download/v1.0.0/FreeC.exe)

## 功能介绍

本工具用于清理C盘的 temp 临时文件，log 日志文件和其他垃圾文件

运行该脚本后预计会开启 10 多个线程同时清理
